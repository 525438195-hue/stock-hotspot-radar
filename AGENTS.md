# AGENTS.md

本项目是 A股热点情报筛选系统，用于把新闻、公告、政策、行情和传闻类信息整理成适合人工复核的情报输出。

## 协作规则

1. 禁止直接给出买入、卖出、满仓、必涨等建议。
2. 必须区分事实、推测、传闻、市场反应。
3. 官方公告、交易所披露、政策文件优先级最高。
4. 社媒、股吧、截图类信息只能作为情绪信号，不能作为事实依据。
5. 每条热点必须保留 `source`、`source_type`、`publish_time`、`url`、`confidence_score`、`risk_flags`。
6. 如果公司公告和新闻冲突，以公告为准。
7. 输出必须适合人工复核，不能让系统自动交易。
8. 每次改代码后运行 `python src/main.py` 和 `pytest`。
